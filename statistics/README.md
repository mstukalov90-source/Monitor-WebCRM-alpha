| [Техническое описание](technical.md) | Разработчики, администраторы БД | Таблицы, триггеры, API, интеграция QGIS и мобильного приложения |

## Версия модели

**Statistics v2** — 4 метрики для поля и 6 для офиса. Запись событий выполняется **триггерами PostgreSQL** при изменениях в CRM-таблицах (WebCRM, QGIS, мобильное приложение пишут в одну БД).

## Быстрый доступ в интерфейсе

Кнопка **«Статистика»** на стартовом экране района (и в шапке приложения). Период по умолчанию — последние 30 дней.
